GUIDE DE DÉPLOIEMENT

Séquence d'initialisation du nœud KORE.

PHASE 01

Exécution du Binaire

Lancez le fichier KORE_Setup_x64.exe que vous venez de télécharger. L'installation est silencieuse et automatique.

PHASE 02

Identification

Au premier lancement, créez votre profil "Pilote". Vos clés SSH seront générées automatiquement pour la sécurisation du tunnel.

PHASE 03

Création d'Instance

Sélectionnez un modpack (ex: "Tech Reborn") et cliquez sur INITIALISER LE NOYAU. Le téléchargement des assets commencera.

PHASE 04

Connexion du Cluster

Une fois le serveur prêt (indicateur vert), cliquez sur PARTAGER LE LIEN. Envoyez ce lien à votre escouade pour qu'ils rejoignent la session.

Dépannage Rapide (FAQ)

Q: Mon antivirus bloque l'exécutable.

R: C'est un faux positif dû à la signature numérique récente. Ajoutez une exception pour KORE.

Q: "Port Binding Error"

R: Une autre instance de Java tourne peut-être en fond. Tuez les processus Java via le Gestionnaire de tâches.
